アプリケーション・オープンソースソフトウェア（OSS）

SimpleController4J

Java Servlet&JSPの開発を支援するシンプルなフレームワークです。
Javaサーブレットにルーティング・データバインディング・アスペクトの機能を追加します。
GPLv3に基づいて配布をしておりますので、自由に改変をしてご使用ください。
SimpleController4J

SimpleORM4J

Javaアプリケーションの為のシンプルなO/Rマッパです。
シンプルなCriteria APIを提供し、簡易な処理でリレーショナルデータベースのデータを Javaオブジェクトして直感的に操作することができます。
GPLv3に基づいて配布をしておりますので、自由に改変をしてご使用ください。
SimpleORM4J

Tasks

JavaFXで実装したタスク管理アプリケーションです。タスク管理に必要な基本的機能を実装しています。
アプリケーションの基本的な使用方法は以下のサイトをご参照ください。
シンプルに日々のタスクを管理出来るアプリケーションですので、是非ご活用ください。
Tasks

PASSOWRD MAKER

パスワードを生成するアプリケーションです。 最大50桁、100個のパスワードを作成することが出来ます。記号を混ぜることも出来ます。
無料で利用が出来ますので、是非ご活用ください。
PasswordMaker

SimpleTemplate

PHPで記述された「シンプル」かつ「軽量」なテンプレートエンジンです。
BSDライセンスにて配布しておりますので、改変および改変後の公開が自由です。また、改変後の公開を必須としませんので、機能拡張し社内テンプレートとしてご使用いただけます。
SimpleTemplate

SimpleGraph

PHPで記述された「シンプル」なグラフライブラリです。SimpleGraphを使用すると、PHPプログラムの配列をライブラリにパラメータとして渡すことで、簡単にグラフを作成することができます。
BSDライセンスにて配布しておりますので、改変および改変後の公開が自由です。
SimpleGraph
